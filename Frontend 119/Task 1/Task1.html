What is HTML?

HTML is the standart markup language for Web pages.

The HyperText Markup Language or HTML is the standart markup language for documents designedto be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.
In HTML each element is made up of 3 parts:

1.The Opening tag
2.The Closing tag
3.The HTML element

You can learn more at the MDN Web Docs
